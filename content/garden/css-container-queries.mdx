Container queries are what I wish media queries had always been. Instead of asking "how wide is the screen?", you ask "how wide is my container?".

This means a card component can rearrange itself whether it's in a sidebar or a full-width layout — without the parent knowing or caring about the component's internal layout decisions.

```css
.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card { display: grid; grid-template-columns: 1fr 2fr; }
}
```

Browser support is solid now. Use it.
